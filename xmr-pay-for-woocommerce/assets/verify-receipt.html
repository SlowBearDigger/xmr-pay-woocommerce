xmr-pay · receipt verifier

Paste a payment receipt below. Verification runs entirely in your browser — nothing is uploaded. A valid receipt proves the merchant signed exactly these payment facts; if it carries tx_proofs, anyone with a Monero node can also confirm it against the chain.

Offline check needs a browser with WebCrypto Ed25519 (recent Chrome / Safari / Firefox). The receipt format and this verifier are open source: github.com/SlowBearDigger/xmr-pay — host this file anywhere, it has no dependencies.
